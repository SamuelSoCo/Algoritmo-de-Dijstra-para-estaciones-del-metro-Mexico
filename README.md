#  Proyecto: Ruta Más Rápida en el Metro de la CDMX (Algoritmo de Dijkstra)

##  Descripción del Proyecto
Este proyecto es una implementación en **Python** del **Algoritmo de Dijkstra** para resolver el problema de la **ruta más rápida (tiempo mínimo)** en la red del *Sistema de Transporte Colectivo (STC) Metro de la Ciudad de México*.

El sistema modela la red de metro como un **grafo ponderado**, donde:
- Las **estaciones** son nodos.
- El **tiempo de viaje** entre estaciones es el peso de cada arista.

El usuario puede ingresar una estación de **origen** y una de **destino**, y el programa calcula y visualiza la **secuencia óptima de estaciones**, así como el **tiempo total estimado** de recorrido.

| Tecnología | Propósito |
|-------------|------------|
| **Python 3.x** | Lenguaje de programación principal. |
| **NetworkX** | Creación, modelado y análisis del grafo; implementación nativa de Dijkstra. |
| **Pandas** | Lectura y manipulación estructurada de los datos desde el archivo CSV. |
| **NumPy** | Soporte numérico para operaciones y manejo de datos. |
| **Matplotlib** | Visualización gráfica de la ruta más corta encontrada. |
